# Planner Log

Every task gets an entry. Read this log before creating a new plan. Update it when done.

## Format
```
### [DATE] Task: <title>
- **Status**: done | in-progress | blocked
- **Plan Quality**: good | needs-work | bad
- **What Went Right**: steps that worked well
- **What Went Wrong**: steps that failed or were missed
```
